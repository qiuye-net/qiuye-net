hexo-blog搭建

学习

2020-09-13

备忘一下我搭建此blog的过程，感谢磊哥手把手教学：）

1、安装node.js/git.exe

-从node.js官网下载最新版本node.exe并安装
-在node.exe同级目录下新建blog文件夹
-从Git官网下载git.exe并安装
以上操作的注意事项详见(安装教程)

2、安装hexo

-打开git bash输入以下命令
npm install -g hexo-cli
-hexo安装完成后，请特别注意看注释
hexo init folder #安装blog的根目录
hexo init blog #当前文件夹下新建blog目录作为blog的根目录
cd
npm install #安装所有依赖包
hexo server

3、更换主题

-下载主题到themes目录下
-修改_config.yml中theme字段为新主题名
hexo clean
hexo server

4、Github托管

-注册Github帐号
-创建repository
-设置ssh连接
git config –global user.name “xxx”
git config –global user.name “xxx”
-生成本地密钥
ssh-keygen -t rsa -C “xxx@qq.com“

5、发布到Github

-在_config.yml末尾添加deploy
type:git
repo:
github:git@github.com:xxx/xxx.github.io.git,master
-执行如下命令完成发布
hexo clean
hexo d
-访问https://xxx.github.io，即可查看blog
